Desenvolveu-se um sistema para ecrãs de larga escala,
fazendo uso de ponteiros laser convencionais como interface de entrada.
Este sistema suporta uso colaborativo.
Uma interface inovadora foi criada para suportar a interacção com tais dispositivos,
baseada no conceito de portões e menus circulares com uma interface não-intrusiva.

Os utilizadores são capacitados para navegar no mundo virtual utilizando um conjunto de
modos de navegação, nomeadamente: na primeira pessoa, modo bússola e modo examinar, assim como
um modo de vôo multimodal.
Podem ser criadas e modeladas formas simples, fazendo uso de um conjunto de ferramentas
de modelação minimalista.
É possível instanciar edifícios a partir de um conjunto de estilos de fachada pré-existentes,
pelo desenho da sua planta e alçado, dando origem a edifícios únicos.
Os edifícios e restantes objectos podem igualmente ser transformados e clonados.
Pode anexar-se a objectos da cena notas de modo a suportar cenários de revisão.
De forma a poder expandir o conjunto de edifícios passíveis de criar, os estilos de fachada
são definidos usando uma gramática XML, permitindo alterar aspecto, telhado e distribuição
dos elementos que compõem as fachadas.
Um processo alternativo de criação de edifícios é proposto, de modo a tirar proveito
do sistema tanto nas primeiras fases como na apresentação de projectos.
