\abstract{
Procura-se com este trabalho suportar um processo de modelação de formas tridimensionais simples
mas capaz de permitir a execução das mais frequentes operações com geometria.
O mesmo assenta num modelo de dados coerente, controlado por uma interface contextual
que permite ao utilizador invocar operações primitivas,
testar a sua aplicação em tempo real e escolher direcções úteis de aplicação
(normal à face, aresta dentro da face vizinha, etc.).
A selecção e aplicação das operações assenta sobre o modelo de interface baseado em traços.
}

\keywords{manipulação directa, modelação tridimensional, interacção por via de gestos.}
