Este projecto é motivado pela vontade de auxiliar uma profissão de importância como é a arquitectura.
A inclusão do projecto no consórcio Improve é igualmente motivante, estabelecendo um conjunto
de objectivos e prazos adicionais ao projecto.

O projecto teve início no final de Setembro.
Houve uma reunião com o consórcio em Lisboa, no INESC Lisboa a que o autor compareceu
e onde foi introduzido aos problemas de desenvolvimento e prazos estipulados.

Houve igualmente um conjunto de reuniões com vista a elucidar os diversos alunos de mestrado quanto
às tarefas que deles se espera e os prazos a cumprir.

Foi criado um blog \cite{SITE-BLOG} para documentar o desenvolvimento do projecto,
de modo a fornecer aos orientadores uma ferramenta que lhes permita acompanhar o mesmo,
assim como documentar as diversas etapas para uso do autor.

Seguiu-se uma fase de preparação para as ferramentas a usar.
Foram executados pequenos programas para ganhar à vontade no uso do OpenSG \cite{SITE-OPENSG}
e instalado todo o software necessário ao desenvolvimento do projecto, entre ele:

\begin{description}
	\item[Microsoft Visual Studio 2003]
		como ambiente de compilação;
	\item[TortoiseSVN]
		para uso de repositório, permitindo o desenvolvimento colaborativo de software;
	\item[Framework AICI]
		conjunto de classes sobre o qual assenta o Improve e que é baseado em OpenSG
\end{description}

Foram efectuadas diversas reuniões com o propósito de definir objectivos para o projecto
e foram escritos casos de uso.
