\section{Conclusões}

Tentou-se com este trabalho disponibilizar a utilizadores
de ecrãs de larga escala uma interface não intrusiva para
navegação tridimensional e manipulação geométrica.
Para tal recorreu-se ao uso de reconhecimento de fala
para alteração do modo de utilização e de um sistema
de reconhecimento de movimentos para fixar os parâmetros
da acção em aplicação.

Houve um esforço em dotar os utilizadores de uma interface
simples, todavia com poder expressivo suficiente para permitir
capacitar os utilizadores das acções mais comuns de manipulação
tridimensional.
Julga-se ter conseguido com estes modos uma curva de aprendizagem
leve, como mostraram os testes efectuados, tendo bastando uma introdução
de 5-10 minutos às acções disponíveis para que os mesmos as efectuassem
de forma expedita.

As operações permitem ao utilizador uma maior imersão no ambiente
virtual, já que não o forçam a interagir com os tradicionais menus, 
que levam a uma mudança de contexto.

Os tempos médios de execução de tarefas de manipulação foram viáveis
(respectivamente 7.4, 10 e 8.36 segundos para as tarefas de drag, rotate e scale.
Para a tarefa de voo o tempo médio foi de 21.65 segundos, um tempo
viável face à navegação num mapa com pouca familiaridade.
A navegação neste modo foi comparada com um outro baseado na escolha
das direcções / orientações básicas face ao ponto de vista, cuja tempo médio
na execução das mesmas tarefas foi de 43.79 segundos. Foi assim uma melhoria de 51\%.

Na navegação para destinos que requeiram um posicionamento vantajoso (leia-se
uma perspectiva abrangente) o gajo de tempo é ainda mais dramático (na ordem dos 70\%),
já que requer uma combinação de direcções e rotações ortodoxa recorrendo ao modo alternativo.

Os principais problemas foram de ordem tecnológica, não inviabilizando de nenhuma forma
os gestos escolhidos mas antes antevendo uma maior facilidade futura de implementação.
O lançamento de consolas de jogos como a Nintendo Wii vieram popularizar estes modos de interacção,
antevendo-se uma banalização dos mesmos e consequente queda de preço.

Notou-se um elevado nível de entusiasmo por parte dos utilizadores
que testaram este sistema, particularmente no modo de voo.
